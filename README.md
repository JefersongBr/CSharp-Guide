# Aprendendo C#

Projeto criado para ajudar pessoas que querem aprender C#.  
Somos estudantes aprendendo juntos e compartilhando conhecimento de forma simples e prática.

Aqui você encontrará conteúdos desde conceitos básicos de lógica de programação até tópicos mais avançados, como APIs e banco de dados.

## O que será ensinado

- Lógica de programação
- Variáveis e tipos
- Condições e loops
- Funções
- Orientação a Objetos
- Estruturas de dados
- Manipulação de arquivos
- Banco de dados
- APIs com ASP.NET
- E muito mais...

Nosso objetivo é evoluir junto com a comunidade e tornar o aprendizado de C# mais acessível para todos.
